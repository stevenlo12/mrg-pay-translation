# mrg-pay-translation

This repository is used to maintain translation file for mrg-pay.com and already integrated with mrg-pay AWS account to manage translation file automatically.


To use this framework, follow these steps:

1. **Clone as Template**:

   ```bash
   git clone https://github.com/FIVEJACK/mrg-pay-translation
   cd mrg-pay-translation
   ```

2. Copy environment variables:

   ```bash
   cp .env.example .env
   ```

3. Install dependencies:

   ```bash
   yarn install
   ```

5. Run Script:

   ```bash
   yarn execute
   ```
